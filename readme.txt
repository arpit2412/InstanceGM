This is the implementation of InstanceGM: Instance-Dependent Noisy Label Learning via Graphical Modelling

We provided the code for implementation on cifar-10 and our code is highly based on DivideMix and CausalNL github implementation. 


For the requirements please install the packages using:

$ pip install -r requirements.txt


For the dataset, please download the cifar10 and have it in the folder named "cifar10"
Download LInk: https://www.cs.toronto.edu/~kriz/cifar.html

For running the code please run it using:

$ python instanceGM.py -r 0.5 

Here r denotes the noise rate.
